# barcode-scanner
<p>
You can scan barcode<br/>
Python 3.8 or higher<br/>
</p>

## Run barcode scanner:
  
```
sudo apt-get install libzbar0
pip install pyzbar
git clone https://github.com/m0hammadhossein/barcode-scanner.git
cd barcode-scanner
python main.py -i <yourimage>
```
